1- Asegurese de tener a nodejs instalado en su equipo, se recomienda una version LTS
2- Puede descargar o clonar el repositorio
3- Una vez descargado o clonado ejecute en la raiz el comando npm install (Este comando te instala las dependencias registradas en el package json)
4- Ejecute el app segun el comando registrado en Packagejson

...Cacharreenlo que lo vamos a modificar a nuestra necesidad.
